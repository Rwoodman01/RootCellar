# Animals Module

Functional alpha module for household-scale animal memory.

V1 supports:

- Animal groups for flocks, herds, colonies, batches, and yards
- Individual animal records for named or tagged animals
- Unified timeline entries for care, health, feed, production, losses, moves, sales, processing, and notes
- Feed records
- Production records
- Simple care reminders with a placeholder field for future Chores integration
- Local-only persistence through the shared Rootcellar localStorage document

This module intentionally avoids commercial farm accounting, diagnosis, deep pedigree tools, AI, backend/auth/payments, and a full Chores engine.
