# Daily Bread and Weekly Huddle

Local-only rhythm layer for Rootcellar.

- Daily Bread gathers today's owned work, due chores, animal care, garden dates, pantry use-soon notes, preservation sessions, and carry-forward work.
- Weekly Huddle reviews Pulse, season priorities, owned work, slipped module signals, stuck items, decisions, and next-week focus.
- Data is persisted inside the existing `rootcellar.alpha.v1` browser localStorage record.
